# Karbon Zero Fes

![GIF](https://github.com/user-attachments/assets/1815a9b6-d47f-43d3-8e33-433f45e9b25d)

This repository contains a customized version of the [Karbon] visualizer, created for [Zero Fes], a music festival for teen artists held on March 29, 2026. It was used in a live performance by [BabyChiefDoit].

[Karbon]: https://github.com/keijiro/Karbon
[Zero Fes]: https://idear.zaiko.io/e/zerofes2026
[BabyChiefDoit]: https://en.wikipedia.org/wiki/BabyChiefDoit

Most parts of the program remain unchanged, but I modified the pixel layout, video clips, and some effect parameters.
